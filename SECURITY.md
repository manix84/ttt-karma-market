# Security Policy

## Reporting a Vulnerability

TTT Karma Market is a Garry's Mod TTT addon. It does not use external services
or persistent storage by default.

If you find a security issue, please report it privately through GitHub's
security advisory flow if available, or contact the maintainer through the
repository or Steam Workshop page. Please avoid posting exploit details publicly
until there has been time to assess and fix the issue.

Useful details to include:

- The affected addon version or commit.
- The Garry's Mod and TTT environment where it was reproduced.
- Steps to reproduce the issue.
- Any relevant console errors or server logs.

## Scope

Security reports should focus on issues that could affect a server, players, or
Workshop subscribers, such as privilege bypasses, unsafe networking behavior,
or server/client crashes caused by addon code.
