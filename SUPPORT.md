# Support

For help with TTT Karma Market, use the channel that best fits the issue:

- Steam Workshop comments for player/server-owner installation and gameplay questions.
- GitHub issues for reproducible bugs, configuration problems, or code changes.
- GitHub Releases for packaged manual install downloads.

When reporting a problem, include the addon version, whether it came from Steam
Workshop or a GitHub Release, and any relevant server/client console errors.

Please do not use public issues or comments to disclose security-sensitive
details. See [SECURITY.md](SECURITY.md) for vulnerability reports.
